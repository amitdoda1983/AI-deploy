This is a readme file.

Do the following steps locally in your own environment.
1. Iris_model_creation.ipynb : code file to build our own model
2. DEMO-local-xgboost-model : learnt model file

Start AWS sagemaker studio --> Notebook instances --> Start Jupyter notebook and run this code on it. Upload the model file and the test data file.
3. my-ml-deploy-aws.ipynb : code to be executed on AWS sagemaker for model deployment, endpoint creation, and evaluation
4. test_point.csv : test data to be used for evaluation
